A simple Android application that shows all the official Brandenburg bathing areas on a map and provide a details screen for them. It also has some extra feature like: favorites, filter based on available services (parking, lifeguard, â€¦), navigation to the location and more.

#### How to install it
The application is not yet available in the play store. However an apk for all the versions can be downloaded in the [relase](../../releases) section and installed manually.
